The Wheel of Creation:

Monster Generator

This is a tool for generating monsters for 5th Edition Dungeons & Dragons, themed around my upcoming setting supplement, the Wheel of Creation. It produces markdown files that you can copy and paste into tools like the Homebrewery or GM Binder to produce beautiful statblocks, styled like those in the official D&D books.

  1. Select a Statblock to start with.
  2. Choose a tag table. Physical tags typically change a monster's statistics, while mental tags are usually only to help the GM/DM determine how it might behave. Tag and Effect describe how the tag will change the basic Statblock. Weight is the chance a tag will be selected by Add Random.
  3. Click on tags in the table to add them, or hit Add Random. They'll show up below.
  4. Click apply.
  5. Copy the text at the bottom and paste it into a document on GM Binder or the Homebrewery. You've got a modified monster!
Notes:
  • You can select tags from multiple tables.
  • Tags are applied left-to-right.
  • Most tags don't stack (that is, they have no effect when applied more than once). These won't be added by the Add Random button.
  • Some tags are mutually exclusive, like 'Wintry' and 'Springy' in 'Fey Physical'. Only the last tag to be applied will have any effect.
